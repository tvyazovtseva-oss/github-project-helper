

# План: Расширение портала мамы по карте сайта

## Текущее состояние vs Требуемое

| Раздел по карте | Сейчас | Нужно |
|---|---|---|
| **Главная** — баннер, лента, медкарта (кратко), профиль малыша, статус подписок | Баннер + quick actions + лента новостей | Добавить: медкарта-аккордеон, профиль малыша, статус подписок/курсов |
| **Клубы/Курсы** — список продуктов, страница продукта с программой и материалами | Только picker в хедере | Новая страница `/mama/courses` + `/mama/courses/:id` |
| **Медкарта (полная)** — прививки, анализы, чекапы, замеры, заметки | Разбросано по табам в Profile | Отдельная страница `/mama/health` с 5 секциями |
| **Библиотека** | Есть (поиск + фильтры + список) | Добавить фильтр по типу (вебинар/гайд), статус просмотра |
| **Уведомления** | Sheet в Layout | Отдельная страница `/mama/notifications` |
| **Профиль** — мама, дети, настройки, подписки | Табы в MamaProfilePage | Расширить: список детей, настройки уведомлений, управление подписками |

## Этапы реализации

### 1. Переработать MamaHomePage
- Персональный баннер с именем мамы/ребенка и активным продуктом
- Глобальная лента обновлений (события по всем продуктам с действиями: открыть, скрыть, отметить прочитанным)
- Медкарта (краткий вид) — аккордеоны по блокам (прививки, анализы, чекапы, прогресс) с переходом в полную медкарту
- Блок профиля малыша (имя, возраст, ключевые параметры, прогресс заполнения)
- Статус подписок/курсов (список активных продуктов с переходом)

### 2. Создать страницы Клубы/Курсы
- **MamaCoursesPage** (`/mama/courses`) — список всех купленных продуктов (название, тип, статус, прогресс)
- **MamaCourseDetailPage** (`/mama/courses/:id`) — хедер продукта (обложка, описание), программа (модули/уроки со статусом), материалы (видео, гайды, файлы)

### 3. Создать полную Медкарту
- **MamaHealthPage** (`/mama/health`) — 5 секций:
  - Прививки (название, возраст, статус, дата — действия: отметить, запланировать)
  - Анализы (тип, периодичность, статус — действия: отметить, добавить результат)
  - Чекапы/осмотры (рекомендованные, возраст, статус)
  - Замеры роста/веса (история + добавить/редактировать/удалить) — перенести GrowthChart сюда
  - Особые заметки (текст + теги)

### 4. Создать страницу Уведомлений
- **MamaNotificationsPage** (`/mama/notifications`) — полный список уведомлений с источниками (курс/медкарта), статусом прочтения, переходом к связанному экрану

### 5. Расширить Библиотеку
- Добавить фильтр по типу материала (вебинар/гайд/файл)
- Добавить фильтр по продукту
- Добавить статус просмотра (новое/просмотрено)

### 6. Расширить Профиль
- Профиль мамы (редактирование имени, аватара, email)
- Профили детей (список детей, выбор активного, добавление нового)
- Настройки уведомлений (push/email, вкл/выкл по типам)
- Управление подписками (список, статусы, продление/отмена)

### 7. Обновить навигацию
- Обновить bottom-nav в MamaLayout: Главная, Курсы, Медкарта, Библиотека, Профиль
- Добавить новые маршруты в App.tsx
- Уведомления — доступны через иконку колокольчика в хедере → переход на `/mama/notifications`
